CartaTypes

Description

En este modulo se describen los tipos bases para describir un elemento de una carta de restaurante. Por simplicidad se ha definido que un elemento de carta tiene nombre, posiblemente descripcion y al menos un precio. Para otro tipo de elementos con mas precios como para describir un trago en la seccion de licores se ha dispuesto dos tipos mas que pueden tener dos o tres precios.
